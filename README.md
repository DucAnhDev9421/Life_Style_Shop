# Life_Style_Shop

## Quick start

### Backend
```bash
cd backend
cp .env.example .env
npm install
npm run dev
```

Backend runs at `http://localhost:5000`.

### Frontend
```bash
cd frontend
cp .env.example .env
npm install
npm run dev
```

Frontend runs at `http://localhost:5173`.

## Included setup

- Frontend: React + Vite + React Router + Axios + Redux Toolkit + Tailwind CSS
- Backend: Express + CORS + Morgan + dotenv + nodemon
- Base API prefix: `/api/v1`
